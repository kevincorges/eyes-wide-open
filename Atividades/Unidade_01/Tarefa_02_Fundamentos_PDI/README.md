# Atividade 2: Fundamentos de Processamento de Imagens Digitais

Este projeto faz parte da Unidade 1 da capacitação em Visão Computacional. O objetivo foi construir um mini-dataset próprio e analisar o impacto técnico de diferentes transformações de imagens.

## 🚀 Objetivos Alcançados
- **Aquisição Controlada:** Captura de 10 imagens originais (classes: Flor e Folha) com variações de iluminação e ângulo.
- **Análise de Resolução:** Estudo do impacto da redução de resolução (50% e 20%) na preservação de detalhes para modelos de CV.
- **Espaços de Cor:** Comparação entre RGB, HSV e Escala de Cinza para tarefas de segmentação.
- **Quantização:** Implementação de redução de níveis de cinza para análise de gradientes e contornos.
- **Formatos de Arquivo:** Análise comparativa de peso e perda de dados entre JPEG e PNG.

## 🛠️ Tecnologias Utilizadas
- Python
- OpenCV
- Matplotlib / Numpy
- Google Colab
